# Dashboard de Anúncios de Veículos

## Descrição do projeto

Este projeto consiste em um aplicativo web interativo desenvolvido com Streamlit para análise exploratória de dados de anúncios de veículos.

O usuário pode visualizar e filtrar os dados, além de explorar gráficos que ajudam na compreensão das informações.

## Funcionalidades

* Visualização dos dados
* Filtro por preço
* Filtro por quilometragem
* Histograma da quilometragem
* Gráfico de dispersão (preço vs quilometragem)

## Tecnologias utilizadas

* Python
* Pandas
* Plotly Express
* Streamlit

## Como executar localmente

1. Clone o repositório

2. Crie um ambiente virtual

3. Instale as dependências:
   pip install -r requirements.txt

4. Execute o app:
   streamlit run app.py
